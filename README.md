# Autonomous Vehicle Development - BFMC

## Project Description
This repository contains the source code and development files for an autonomous robot car designed for the Bosch Future Mobility Challenge (BFMC). The project focuses on building a self-driving model capable of lane keeping, object detection, and autonomous navigation.

## Key Features & Technologies
* **Framework & Simulation:** ROS, Gazebo environment.
* **Control Algorithms:** Model Predictive Control (MPC) algorithm for precise path-following.
* **Perception:** Integrated object detection and travel route synchronization.
* **Programming Languages:** Python and C++.

## Verification & Testing (QA Focus)
* **Real-World Testing:** Conducted direct validation and performance testing on a physical robot car model to ensure the control algorithms operate accurately in real environments.
* **Regression Testing:** Performed regression test cases after integrating the perception (object detection) module with the control system to ensure no functional degradation.
* **System Troubleshooting:** Identified and resolved integration defects between software algorithms and hardware components during deployment.
